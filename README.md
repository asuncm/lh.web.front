# websites-app

### 项目说明

* 本项目为single-spa微服务主应用
* 项目采用react框架开发，react-router作为主体路由分发器

### 环境安装

* npm install
* yarn install
* pnpm install

### 开发依赖包安装
* npm install --save-dev *
* yarn add --dev *
* pnpm add -D *

### 依赖性包安装
* npm install --save *
* yarn add *
* pnpm add *
